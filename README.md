# 🏙️ 新竹 3D 城市迷宮 (Hsinchu 3D City Maze - Final Edition)

這是一個以新竹市為背景，由真實 OSM 數據驅動的 3D 城市探索遊戲，支援手機與 PC 跨平台體驗。

## 🚀 核心更新：極致效能與沉浸體驗

### 1. 空間網格雜湊優化 (Spatial Grid Hashing)
- **海量數據處理**：針對新竹市區超過 15,000 條道路進行了空間分區索引。
- **千倍效能提升**：系統每幀僅計算玩家周邊 100m 內的物理判定，徹底解決大範圍地圖導致的 CPU 卡頓與按鍵延遲。

### 2. 跨平台操作與引導系統
- **PC 使用者**：
  - **移動**：支援 `WASD` 或 `方向鍵`。
  - **旋轉**：全螢幕「按住滑鼠左鍵拖曳」即可旋轉世界。
  - **縮放**：使用「滑鼠滾輪」調整視角。
- **手機使用者**：
  - **移動/旋轉**：左右分區固定操作墊，避開手機邊緣防止誤觸系統返回手勢。
- **操作指南**：點擊右上角 `i` 按鈕可隨時呼叫跨平台操作引導彈窗。

### 3. 地標與植被系統 (Landmarks & Oasis)
- **架空地標**：新增「光華迷宮」、「新竹驛」、「巨大城堡」等奇幻代稱地標，具備固定世界朝向。
- **隨機植被**：在起點區域自動生成 1,500 棵高效能數位樹木 (Instanced Rendering)，增加環境豐富度。

### 4. 專業級渲染與導航
- **對數深度緩衝**：解決 8km 尺度下的 Z-fighting 破圖問題。
- **立體路面**：0.2m 厚黑色柏油路與明亮綠地形成高對比視覺。
- **十字指北針**：全方位 N-S-E-W 實時聯動。
- **智慧 HUD**：自動顯示當前路名，無名區域自動隱藏。

## 📂 技術規範
- **座標系**：標準 3D 座標系（North = -Z, East = +X, Up = +Y）。
- **數據源**：OpenStreetMap (Overpass API)。
- **起點**：精確定位於新竹市光華區 [120.974, 24.817]。

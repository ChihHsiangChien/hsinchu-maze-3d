# 🏙️ 新竹 3D 城市迷宮 (Hsinchu 3D City Maze - Guanghua Edition)

這是一個基於 **Three.js** 與 **OpenStreetMap (OSM)** 數據驅動的 3D 城市探索遊戲。玩家將在新竹市中心真實的地圖數據中，體驗 2.5D 第三人稱視角的探索。

## 🚀 核心更新：光華迷宮與地標系統

### 1. 座標與方位校正 (True Alignment)
本專案已完成最終方位校正，達成視覺、物理與操作的 1:1 同步：
- **北方 (North)** = 3D 世界 **-Z 方向** (符合 Three.js 前方標準)。
- **東方 (East)** = 3D 世界 **+X 方向** (符合地理直覺)。
- **映射公式**：`x = lon_offset`, `z = -lat_offset`。配合 `rotateX(-Math.PI/2)` 旋轉，確保東西向無鏡像翻轉。

### 2. 自定義地標資料庫 (Landmarks System)
引入了 `public/data/landmarks.json` 設定檔，可大規模建置 3D 路牌：
- **固定朝向**：路牌採用 3D Mesh 實體化，具備固定世界旋轉角（而非自動正對相機），模擬真實街景。
- **架空地名**：為了增加探索樂趣，我們為新竹地標設定了奇幻代稱，如「巨大城堡」(Big City)、「新竹驛」(車站)、「幽冥法庭」(城隍廟)。

### 3. 視覺優化
- **城市峽谷**：建築物高度提升至主角身高的 5 倍（約 8~10 米），強化迷宮空間感。
- **高對比色彩**：深黑色立體路面 (0.2m 厚) 與明亮嫩綠色草地，徹底解決 Z-fighting 閃爍並提升可讀性。

## 🎮 操作說明

- **右手搖桿區**：控制角色位移（向上推 = 朝主角正面走，向右推 = 向右平移）。
- **左手旋轉區**：水平滑動可旋轉世界方位角（繞著主角垂直 Y 軸）。
- **指北針**：右上角紅色針尖與 "N" 標籤隨世界同步旋轉，永遠指向真實北方 (-Z)。

## 📂 資料結構
- `public/data/roads.json`: 清洗過的 OSM 道路多邊形數據。
- `public/data/landmarks.json`: 自定義 3D 地標設定檔。
- `src/core/Player.js`: 處理 2.5D 第三人稱追隨、滑動碰撞與座標標籤。

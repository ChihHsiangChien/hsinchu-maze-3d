# 🏙️ 新竹 3D 城市迷宮 (Hsinchu 3D City Maze - Cross-Platform Edition)

這是一個以新竹市為背景，由真實 OSM 數據驅動的 3D 城市探索遊戲，支援手機與 PC 跨平台體驗。

## 🚀 核心更新：全平台優化與視覺穩定

### 1. 跨平台操作系統 (Unified Controls)
- **PC 使用者**：
  - **移動**：支援 `WASD` 或 `方向鍵`。
  - **旋轉**：在螢幕任意處「按住滑鼠左鍵拖曳」即可旋轉世界。
  - **縮放**：使用「滑鼠滾輪」調整視角遠近。
- **手機使用者**：
  - **移動**：右側固定搖桿區（向上推前進）。
  - **旋轉**：左側固定轉向區（水平滑動）。
  - **安全設計**：操作區避開手機邊緣，防止誤觸系統返回手勢。

### 2. 專業級渲染優化 (Visual Stability)
- **解決破圖**：啟用 `logarithmicDepthBuffer` (對數深度緩衝)，確保在 8km x 8km 的大規模場景下，遠處建築與道路不會產生 Z-fighting 閃爍。
- **高對比視覺**：明亮綠地與 0.2m 厚的立體黑色柏油路，提供極佳的路徑辨識度。

### 3. 智慧導航介面
- **十字指北針**：右上角方位盤實時連動，紅色 N 指標永遠指向北方。
- **自動化 HUD**：走在有名字的街道時自動顯示路名，進入無名區域時自動隱藏，釋放視野。

## 📂 技術規範
- **座標系**：採用標準遊戲座標系（North = -Z, East = +X, Up = +Y）。
- **數據源**：OpenStreetMap (Overpass API)。
- **引擎**：Three.js + Turf.js。

# 🏙️ 新竹 3D 城市迷宮 (Hsinchu 3D City Maze - Multi-User Edition)

這是一個以新竹市為背景，由真實 OSM 數據驅動的多人 3D 城市探索遊戲。支援手機與 PC 跨平台連線，具備中央控管系統與智慧機器人模擬。

## 🚀 快速啟動

### 1. 修改設定檔 (選配)
編輯 `public/data/settings.json` 可自訂出生地點、伺服器埠口、機器人數量等。

### 2. 啟動多人連線後端 (Socket.io)
```bash
node server.js
```
後端會依據設定檔自動監聽對應埠口（預設 `8888`）。

### 3. 啟動前端開發伺服器 (Vite)
```bash
npm run dev
```
前端運行於 `5173` 埠。可透過 `http://localhost:5173` 存取。

### 4. 啟動模擬機器人 (選配)
```bash
node scripts/simulate_bots.js
```
依據設定檔生成指定數量的智慧導航機器人。

## ⚙️ 中央設定化系統 (`settings.json`)
本專案採用中心化管理，修改 `public/data/settings.json` 即可同步更新全系統：
*   `origin`: 地圖投影原點 `[緯度, 經度]`。
*   `spawn`: 玩家與機器人出生點 `[緯度, 經度]`。
*   `serverPort`: 系統通訊埠口。
*   `botCount`: 測試機器人數量。
*   `treeCount`: 環境樹木生成總數。
*   `fogDistance`: 迷霧能見度（影響遠景渲染效能）。

## 🕹️ 遊戲模式說明

### 🔵 玩家模式 (Player Mode)
*   **視覺設計**：採用簡約可愛的「浮空頭像」設計，移除身體以優化效能。
*   **水汪汪大眼**：角色具備層次感的眼白與黑瞳，眼距寬大呆萌。
*   **動態標籤**：姓名標籤位於頭頂 4.5 米高處，隨距離自動縮放並補償高度，確保遠處亦可辨識。
*   **視距剔除**：僅渲染周邊 600 米內的隊友，極大化行動裝置運算效能。

### 🔴 管理員模式 (Admin Mode)
*   **上帝視角**：完全隱身，不受地圖邊界限制，具備 4000 米超遠視距。
*   **相機鎖定**：強制限制視角於地平面以上，防止穿透地底。
*   **N 鍵指北**：按鍵盤 `N` 鍵快速重設為正北朝前，同時保留當前縮放與仰角。
*   **智慧召喚**：傳送功能支援「馬路吸附」，自動將玩家安置於最近的街道。
*   **雷達系統**：專屬的亮黃色發光環與高空定位柱，追蹤目標時變更為**電光粉紅**。

## 🛠️ 開發備註
*   **座標格式**：`landmarks.json` 與 `settings.json` 均已統一支援 Google Maps 格式：**`[緯度, 經度]`**。
*   **旋轉邏輯**：`rotation: 0` 代表正面朝向南方 (+Z)，逆時針旋轉。
*   **效能優化**：內建網路節流 (15Hz)、UI 更新防抖與自動記憶體釋放機制。

# 新竹 3D 城市迷宮 (Hsinchu 3D City Maze)

這是一個以新竹市為背景，由真實 OSM 數據驅動的 3D 城市探索遊戲。支援手機與 PC 跨平台操作，具備單機探索與多人連線模式。

## 快速啟動

### 1. 本地開發環境 (需開啟 2-3 個終端機)

若要進行完整的本地多人開發測試，請分別開啟多個終端機視窗執行以下指令：

**終端機 A：啟動前端 (Vite)**
```bash
# 提供網頁前端內容
npm run dev
```
前端運行於 5173 埠。存取地址：http://localhost:5173。

**終端機 B：啟動後端 (多人遊戲伺服器)**
```bash
# 負責處理玩家位置與通訊同步
node server.js
```
預設監聽 8888 埠。

**終端機 C：啟動模擬機器人 (選配)**
```bash
# 生成智慧導航機器人進行壓力測試
node scripts/simulate_bots.js
```
依據設定檔生成指定數量的智慧導航機器人。

## 遠端部署與線上遊玩

### 1. 快速部署到 GitHub Pages
如果您修改了地圖資料（如 landmarks.json）或程式碼，只需在專案根目錄執行以下指令即可完成線上更新：
```bash
npm run deploy
```
該指令會自動完成打包、清理並將 dist/ 內容推送到 gh-pages 分支。

### 2. 線上遊玩模式 (Single vs. Multi)
*   單人模式 (預設)：直接造訪 https://chihhsiangchien.github.io/hsinchu-maze-3d/。系統偵測到 GitHub 環境且無參數時，會自動關閉連線模組以避免報錯。
*   多人模式 (手動指定)：若要讓遠端玩家連回您的伺服器，請在網址後加上 server 參數：
    https://chihhsiangchien.github.io/hsinchu-maze-3d/?server=您的公網IP

> 重要提示：若要在 HTTPS (GitHub) 下連線到不具備 SSL 的 HTTP 伺服器，請在瀏覽器網址列左側點擊「鎖頭/網站設定」，將「不安全的內容 (Insecure content)」設為「允許」。

## 中央設定化系統 (settings.json)
修改 public/data/settings.json 即可同步更新全系統：
*   origin: 地圖投影原點 [緯度, 經度]。
*   spawn: 玩家與機器人出生點 [緯度, 經度]。
*   serverHost: 預設伺服器位址 (如 localhost)。
*   serverPort: 伺服器通訊埠口 (預設 8888)。
*   botCount: 測試機器人數量。
*   treeCount: 環境樹木生成總數。
*   fogDistance: 迷霧能見度（影響遠景渲染效能）。

## 遊戲操作與設計

### 玩家模式 (Player Mode)
*   視覺設計：採用簡約可愛的「浮空頭像」設計，移除身體以優化效能。
*   水汪汪大眼：角色具備層次感的眼白與黑瞳，眼距寬大呆萌。
*   動態標籤：姓名標籤位於頭頂 4.5 米高處，隨距離自動縮放並補償高度，確保遠處亦可辨識。
*   視距剔除：僅渲染周邊 600 米內的隊友，極大化行動裝置運算效能。

### 管理員模式 (Admin Mode)
*   上帝視角：完全隱身，不受地圖邊界限制，具備 4000 米超遠視距。
*   相機鎖定：強制限制視角於地平面以上，防止穿透地底。
*   N 鍵指北：按鍵盤 N 鍵快速重設為正北朝前，同時保留當前縮放與仰角。
*   智慧召喚：傳送功能支援「馬路吸附」，自動將玩家安置於最近的街道。
*   雷達系統：專屬的亮黃色發光環與高空定位柱，追蹤目標時變更為電光粉紅。

### 操作方式
*   電腦 (PC)：W/A/S/D 或方向鍵移動，滑鼠拖曳旋轉視角。
*   手機 (Touch)：右下角搖桿移動，左下角感應區旋轉視角。
*   管理員：在網址加入 ?admin=true 進入上帝視角，具備管理員監控面板。

## 開發技術備註
*   座標格式：landmarks.json 與 settings.json 均已統一支援 Google Maps 格式：[緯度, 經度]。
*   旋轉邏輯：地標使用弧度 (Radians) 單位，2π (約 6.28) 為一圈，採逆時針旋轉。
    *   0: 正面朝向南方 (+Z)
    *   1.57 (π/2): 東方 (+X)
    *   3.14 (π): 北方 (-Z)
    *   4.71 (3π/2): 西方 (-X)
    *   方位對照：東南 (0.79)、東北 (2.36)、西北 (3.93)、西南 (5.50)。
*   效能優化：內建網路節流 (15Hz)、UI 更新防抖與自動記憶體釋放機制。

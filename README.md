# 🏙️ 新竹 3D 城市迷宮 (Hsinchu 3D City Maze - Oasis Edition)

這是一個以新竹市為背景，由真實 OSM 數據驅動的 3D 探索沙盒。

## 🚀 核心更新：光華森林與極簡導航

### 1. 高效隨機植被系統 (Cyber Oasis)
- **物理感知**：系統會自動避開黑色路面，在綠地與建築空隙間種植樹木。
- **實例化渲染 (Instanced Rendering)**：利用 `InstancedMesh` 技術，即便畫面上擁有數千棵樹，依然僅佔用 1 個 Draw Call，確保行動端效能極致流暢。

### 2. 3D 地標標牌系統 (Landmarks Database)
- **架空地名**：採用奇幻風格命名新竹地標，如「巨大城堡」(Big City)、「新竹驛」(火車站)、「幽冥法庭」(城隍廟)。
- **固定世界方向**：地標牌採用 3D Mesh 實體化，具備固定的旋轉角，模擬真實路牌手感。
- **擴充性**：透過修改 `public/data/landmarks.json` 即可大規模建置自定義地標。

### 3. UX 與視覺構圖優化
- **電影視角**：主角定位在螢幕下方 1/3，大幅增加前方路徑的預覽範圍。
- **智慧 HUD**：路名顯示框具備自動感應功能，當主角處於無名區域時，HUD 會自動隱藏，提供 100% 純淨視野。
- **滑動碰撞**：支援順著路緣滑行，解決卡頓感。

## 🎮 操作說明
- **右手區**：移動（向上推前進，向右推右移）。
- **左手區**：水平滑動旋轉世界方位（繞主角垂直軸）。
- **導航**：觀察右上角十字指北針（紅色 N 指向北方 -Z）。
